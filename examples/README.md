# Examples

This folder contains example HTML files that you can use to test the Website Content Rewriter.

## Sample Files

- **sample.html**: A sample HTML file with placeholder content and images

## Usage

1. Upload any HTML file from this folder to the Streamlit app
2. Configure your settings
3. Process the file
4. Download the processed result

## Note

You can use your own HTML files - they don't need to be in this folder. This is just for reference and testing.
